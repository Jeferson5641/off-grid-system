<summary><strong> Português (Brasil)</strong></summary>

Uma aplicação web moderna para **dimensionamento de sistemas fotovoltaicos off-grid**. Calcula automaticamente painéis solares, baterias e componentes necessários para sistemas de energia renovável autossuficientes.

O **Off-Grid System Calculator** é uma ferramenta essencial para engenheiros, instaladores e entusiastas de energia renovável. Integra dados reais da **NASA POWER API** para calcular o potencial solar e dimensiona automaticamente todos os componentes.


### ✨ Funcionalidades

✅ **NASA POWER API** - Dados climáticos reais de irradiação solar  
✅ **Cálculo Inteligente** - Dimensiona painéis baseado no consumo  
✅ **Múltiplas Baterias** - LiFePO4, Chumbo-ácido estacionária e automotiva  
✅ **Autonomia** - Calcula dias de funcionamento sem geração  
✅ **Interface Responsiva** - Design moderno com Tailwind + shadcn/ui  
✅ **Multilíngue** - PT/EN/ES com detecção automática

### Como Começar

```bash
git clone https://github.com/Jeferson5641/off-grid-system.git
cd off-grid-system
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000)

### Como Usar

1. Insira Latitude e Longitude
2. Defina o consumo mensal (kWh)
3. Clique para buscar radiação solar (NASA POWER)
4. Configure potência do painel, autonomia e tensão
5. Escolha o tipo de bateria
6. Receba os resultados (painéis, baterias necessárias)

### ✨ Features

✅ **NASA POWER API** - Real climate data for solar irradiation  
✅ **Smart Calculation** - Sizes panels based on consumption  
✅ **Multiple Batteries** - LiFePO4, Stationary and Automotive Lead-acid  
✅ **Autonomy** - Calculates days of operation without generation  
✅ **Responsive Interface** - Modern design with Tailwind + shadcn/ui  
✅ **Multilingual** - PT/EN/ES with automatic detection
